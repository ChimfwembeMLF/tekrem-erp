# Tags

## What is a Tag?
A **Tag** is a label that can be attached to cards to categorize, filter, or highlight work items. Tags help teams organize and find related work quickly.

## Use Cases
- Categorize cards by feature, priority, or team
- Filter boards and backlogs by tag
- Highlight urgent or important work

## User Actions
- Create, edit, and delete tags
- Assign tags to cards
- Filter and search cards by tag

## How Tags Work Together
- Tags provide flexible organization across boards, sprints, and epics
- Tags help teams quickly find and group related work
