# Boards

## What is a Board?
A **Board** is a visual workspace for organizing and tracking work using columns and cards. Boards can represent workflows such as Kanban or Scrum, and are central to visual project management.

## Use Cases
- Visualize project work and status
- Organize cards by workflow stage (e.g., To Do, In Progress, Done)
- Enable drag-and-drop task management
- Support agile methodologies (Kanban, Scrum)

## User Actions
- Create, edit, and delete boards
- Add, edit, and move columns
- Add, edit, move, and delete cards
- Assign cards to users, sprints, epics, or releases
- Filter and search cards by status, assignee, or type
- View board activity and history

## How Boards Work Together
- Boards provide a central view of all project work
- Cards on boards can be linked to sprints, epics, and releases
- Boards help teams visualize progress and bottlenecks
