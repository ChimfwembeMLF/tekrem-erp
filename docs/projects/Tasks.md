# Tasks

## What is a Task?
A **Task** is a specific, actionable work item that needs to be completed as part of a card, story, or project. Tasks can be standalone or part of a checklist within a card.

## Use Cases
- Break down cards into smaller steps
- Assign tasks to team members
- Track progress on detailed work

## User Actions
- Create, edit, and delete tasks
- Assign tasks to users
- Mark tasks as complete or incomplete
- Organize tasks within cards or as standalone items

## How Tasks Work Together
- Tasks help break down complex work into manageable steps
- Completion of tasks contributes to card and project progress
