# Files

## What is a File?
A **File** is any document, image, or attachment uploaded to a card, project, or board. Files help teams share resources, requirements, and deliverables.

## Use Cases
- Attach specifications, designs, or assets to cards
- Share documents with team members
- Store project deliverables and reference materials

## User Actions
- Upload, download, and delete files
- Attach files to cards, boards, or projects
- Organize files by type or context

## How Files Work Together
- Files are linked to cards, boards, or projects for easy access
- Centralized file management supports collaboration and documentation
