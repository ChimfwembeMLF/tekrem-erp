# Milestones

## What is a Milestone?
A **Milestone** is a significant point or event in a project timeline, often representing the completion of a major deliverable or phase. Milestones help track progress toward key goals.

## Use Cases
- Mark important deadlines or achievements
- Track project phases (e.g., MVP, Beta, Launch)
- Align team and stakeholders on progress

## User Actions
- Create, edit, and delete milestones
- Set milestone name, description, due date, and status
- Link milestones to sprints, releases, or epics
- View all work associated with a milestone

## How Milestones Work Together
- Milestones provide high-level checkpoints in the project
- Sprints, epics, and releases can be aligned to milestones
- Milestones help teams and stakeholders visualize progress and upcoming goals
